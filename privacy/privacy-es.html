Politica de privacidad

Esta politica explica como Bet Tracking recopila, usa y protege los datos personales de los usuarios.

Datos recopilados

  • Datos de identificacion (ej. email) cuando el usuario crea una cuenta o inicia sesion.
  • Datos de uso ingresados por el usuario (ej. apuestas, estadisticas, preferencias).
  • Datos tecnicos (ej. tipo de dispositivo, sistema operativo).

Finalidad del tratamiento

  • Ofrecer funciones de la app y sincronizar datos entre dispositivos.
  • Mejorar la experiencia del usuario y la calidad del servicio.
  • Gestionar suscripciones Premium mediante las tiendas oficiales.
  • Mostrar anuncios a usuarios free (cuando esten disponibles).

Servicios de terceros

Usamos servicios de terceros para ofrecer la app, incluidos:

  • Firebase (Google) para autenticacion y sincronizacion de datos.
  • AdMob (Google) para publicidad en la app.
  • Apple App Store / Google Play para compras dentro de la app.

Conservacion de datos

Los datos se conservan el tiempo necesario para ofrecer el servicio. El usuario puede solicitar la eliminacion de la cuenta en cualquier momento; la eliminacion es inmediata y definitiva.

Seguridad

Aplicamos medidas tecnicas y organizativas para proteger los datos de accesos no autorizados o perdida.
