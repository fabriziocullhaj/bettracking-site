Politica de privacidade

Esta politica explica como Bet Tracking recolhe, usa e protege os dados pessoais dos usuarios.

Dados recolhidos

  • Dados de identificacao (ex. email) quando o usuario cria conta ou faz login.
  • Dados de uso inseridos pelo usuario (ex. apostas, estatisticas, preferencias).
  • Dados tecnicos (ex. tipo de dispositivo, sistema operativo).

Finalidade do tratamento

  • Fornecer funcionalidades da app e sincronizar dados entre dispositivos.
  • Melhorar a experiencia do usuario e a qualidade do servico.
  • Gerir subscricoes Premium pelas lojas oficiais.
  • Mostrar anuncios a utilizadores free (quando disponiveis).

Servicos de terceiros

Usamos servicos de terceiros para fornecer a app, incluindo:

  • Firebase (Google) para autenticacao e sincronizacao de dados.
  • AdMob (Google) para publicidade na app.
  • Apple App Store / Google Play para compras na app.

Retencao de dados

Os dados sao armazenados pelo tempo necessario para fornecer o servico. O usuario pode pedir a eliminacao da conta a qualquer momento; a eliminacao e imediata e definitiva.

Seguranca

Aplicamos medidas tecnicas e organizacionais para proteger os dados de acesso nao autorizado ou perda.

Direitos do usuario

O usuario pode solicitar acesso, correcao ou eliminacao dos seus dados. Para pedidos de privacidade:
